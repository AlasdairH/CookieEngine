Detailed Description

A Singleton for handling the storage of resources.

This class handles the storage of resources for use by elements of the game engine. This allows the same Texture, ShaderProgram, etc to be used by multiple entities or services. The resources are stored in a map and access is checked before operations on the stored resources is allowed. This class may need a bit of refactoring, seems like I'm writing a lot of semi-duplicate code.
