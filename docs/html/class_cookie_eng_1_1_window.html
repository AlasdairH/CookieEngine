Detailed Description

An abstraction of the SDL Window class which can be used to create a window for OpenGL rendering.

This class uses the SDL Window class to create a window in the constructor. By use of default parameters, a default window can be created by passing no arguments to the constructor or various settings can be applied. An OpenGL Context is created with the window for rendering.
